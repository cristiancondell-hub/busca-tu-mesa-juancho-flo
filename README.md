# Busca tu mesa

App estática para buscar mesa por apellido.

## Archivos

- `index.html`: app principal para invitados.
- `app.js`: listado de invitados y lógica de búsqueda.
- `styles.css`: diseño visual.
- `qr.html`: generador del código QR cuando exista la URL final publicada.

## Uso

1. Abrir `index.html` para probar la app.
2. Subir la carpeta completa `busca-tu-mesa` a un hosting estático.
3. Abrir `qr.html`, pegar la URL pública de `index.html` y usar el QR generado para imprimirlo o compartirlo.
